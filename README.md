# jikeCountDown
一个简单的微信小程序，用来查看即刻停服了多久。
我给它取了一个不错的中文名字：倒即时。

## 为什么要做？
原因很简单，我是即刻app的资深用户，很喜欢它。
由于众所周知的原因，即刻的所有服务在2019年7月12日关停。
希望即刻早日回归！
